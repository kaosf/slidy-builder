# Slidy HTML Builder

## Installation

```sh
./configure
# or
# ./configure --prefix=$HOME/local

make install
```

## Usage

```sh
slidybuild < input.md > index.html
```

`index.html`, `slidy.css` and `slidy.js` are destination files.
